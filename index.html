O que é uma tag?

Tags são comandos usados em HTML para estruturar páginas. Clique nos outros cartões para ver exemplos!

<p>

Cria parágrafos de texto.

<h1>

Título principal da página.

<h2>

Subtítulo, menor que h1.

<a>

Cria links para outras páginas.

<img>

Exibe imagens na página.

<ul>

Lista não ordenada (com pontos).

<ol>

Lista ordenada (numerada).

<li>

Item dentro de listas.

<div>

Bloco genérico para agrupar conteúdo.

<span>

Marca parte do texto em linha.

<strong>

Deixa o texto em negrito.

<em>

Deixa o texto em itálico.

<br>

Quebra de linha.

<hr>

Linha horizontal de separação.

<table>

Cria uma tabela.

<tr>

Linha dentro da tabela.

<td>

Célula de dados da tabela.

<th>

Célula de cabeçalho da tabela.

<form>

Cria um formulário.

<input>

Campo de entrada de dados.
